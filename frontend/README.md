# AI Trading Bot Frontend

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```

## Running the App
1. Start the development server:
   ```bash
   npm start
   ```
   or
   ```bash
   ng serve
   ```
2. Open [http://localhost:4200](http://localhost:4200) in your browser.

## Connection
The frontend connects to the backend API at `http://localhost:8080/api/trades`.
Ensure the Java backend is running.
